Module II - Using NuvlaBox (Basic)
----------------------------------

NuvlaBox a cloud-in-a-box that's child play to use. 
This model lists the prerequisites for working with NuvlaBox and
then guides you from unpacking to your first application deployment via SlipStream.

This module presumes familiarity with cloud technologies, NuvlaBox concept and with the key
concepts of SlipStream.  Review :doc:`Module I <module-1>` if this is
not the case.
